 ⚔ Dungeon Quest API

A fantasy RPG web application built with **Node.js**, **Express.js**, **HTML**, **CSS**, and **JavaScript**. Fight enemies, collect loot, equip weapons, level up, and manage your inventory through an interactive game interface.

# Tech Stack

- Node.js
- Express.js
- HTML
- CSS
- JavaScript


Open **http://localhost:3000**
